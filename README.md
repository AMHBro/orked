# بيانات أوركيد - عرض ونسخ وبحث

واجهة لعرض وبحث ونسخ بيانات المعلمين والطلاب (الاسم، الكود، الباسورد) لثانوية الأوركيد.

## التشغيل

1. افتح `index.html` في المتصفح (نقرة مزدوجة أو Live Server).
2. البيانات تُحمّل من `data.json` أو من البيانات المضمّنة في الصفحة؛ الإضافات والاستيراد يُحفظان في **localStorage**.

## المميزات

- **قسم المعلمين** و**قسم الطلاب** مع بحث ونسخ (خلية / سطر / الكل).
- **استيراد البيانات**: الصق النص (باسورد، كود، اسم) واضغط «استيراد واستبدال».
- **إضافة سجل** جديد (معلم أو طالب).
- فلتر عرض: الكل / المعلمون فقط / الطلاب فقط.

## تحديث البيانات المضمّنة في الصفحة

بعد تعديل `data.json` شغّل:

```bash
node embed-data.js
```

ليتم دمج المحتوى الكامل في الصفحة (مفيد عند فتحها عبر `file://`).

## الملفات

- `index.html` — الصفحة والواجهة والمنطق
- `data.json` — مصدر البيانات (معلمون + طلاب)
- `embed-data.js` — سكربت لدمج البيانات في الصفحة
- `parse-pdf-students.js` — مساعد تحليل نص من PDF (اختياري)
